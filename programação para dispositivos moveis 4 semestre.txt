Plataformas de desenvolvimento para dispositivos móveis:

	* Nativa
		IOS -> Objective-C ou Swift
		Android -> C, Java ou Kotlin

		Xamarim -> escreve em uma linguagem intermediária, e o código é transformado para Objective-C ou Java de acordo com a plataforma

	* Hibrida
		Código feito em HTML, CSS e JS é executado tanto no IOS quanto no Android
		O dispositivo móvel para rodar este código ele abre um navegador

		Framework chamado ==> Phonegap e Apache Cordova

	* Frameworks Nativos/Hibridos

React Native, códigos são feitos em JavaScript => JS5  EcmaScript => ES6
			IOS => Webkit (processador de Javascript)
			Android => V8 (processador de Javascript)

		Dart e Flutter, códigos são feitos em Dart => há um processador de linguagem dart para o IOS e Android
